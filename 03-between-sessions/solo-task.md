# Between Sessions: Solo Task [WIP]

This takes about 10 minutes. Do it before Session 2.

---

## The task

1. Open your project folder in VSCode
2. Start Claude Code in the terminal:
   ```
   claude
   ```
3. Ask it this question:
   ```
   What does the manifest.json file do in a Figma plugin? Explain it simply.
   ```
4. Read the answer.
5. Open your `README.md` file and add a section called `## Notes` at the bottom.
6. Write 2–3 sentences summarizing what CC told you — in your own words.
7. Save the file, commit it:
   ```
   git add README.md
   git commit -m "add notes on manifest.json"
   git push
   ```

---

## Why this matters

You just used CC independently, pulled information from it, and committed the result.
That's the loop you'll be doing all day in Session 2 — just with code instead of notes.

If anything in the steps above didn't work, write down what happened and bring it to Session 2.
