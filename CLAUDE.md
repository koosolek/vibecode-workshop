# VibeCode Workshop

This is a workshop facilitation project. It contains session guides, participant
materials, and plugin templates for a 2-session Claude Code onboarding workshop.

## Purpose

Help a team of 3 non-engineers (no CLI, no git, no AI tooling experience) learn
Claude Code by building and publishing a real Figma plugin.

## Structure

- `pre-work/` — send to participants before the workshop
- `session-1/` — facilitator guide for Session 1 (environment + CC fundamentals)
- `between-sessions/` — solo task participants do independently
- `session-2/` — facilitator guide for Session 2 (Figma MCP + plugin build)
- `workshop-plugin/` — pre-seeded into participant repos before Session 2

## Guidelines when working on this project

- Participant-facing content: assume zero CLI, git, or coding experience
- Facilitator guides: practical and step-by-step, not slide decks
- Commands shown in guides must be copy-paste ready and tested
- Keep session timing realistic — buffer for questions and small failures
